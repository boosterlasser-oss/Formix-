So funktionierts

In 3 Schritten zum Ergebnis

Keine komplizierte Einrichtung. Einfach herunterladen und loslegen.

1

Profil anlegen

Gib deine Körperdaten ein. FORMIX berechnet deinen Kalorienbedarf und erstellt dein persönliches Profil.

2

Programm wählen

Wähle aus fertigen Trainingsprogrammen – passend zu deinem Ziel und Fitnesslevel.

3

Trainieren & tracken

Hak Sätze ab, log deine Mahlzeiten und beobachte wie du Woche für Woche stärker wirst.

2. Welche Daten werden verarbeitet?

Die App speichert ausschließlich lokal auf deinem Gerät:

  • Körperdaten (Gewicht, Größe, Alter, Geschlecht)
  • Trainingsdaten und -protokolle
  • Ernährungs- und Kaloriendaten
  • App-Einstellungen

Diese Daten werden nicht an uns oder Dritte übermittelt.

3. KI-Coach (GitHub Models / Azure)

Beim KI-Coach (PRO) werden Anfragen direkt an die GitHub Models API gesendet. Es gelten die Bestimmungen von GitHub und Microsoft. Wir haben keinen Zugriff auf diese Daten. Der Token wird verschlüsselt gespeichert.

4. Google Play Billing

Zahlungen laufen über Google Play Billing. Wir erhalten nur eine anonymisierte Bestätigung. Es gilt die Datenschutzerklärung von Google.
